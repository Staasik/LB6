1) Значение коэффициента a равно 0 (a=0, b!=0, c!=0). Ожидается вывод одного корня.
2) Значения коэффициентов a и b равны 0 (a=0, b=0, c!=0). Ожидается сообщение Нет корней.
3) Значения всех коэффициентов равны 0 (a=0, b=0, c=0). Ожидается сообщение R.
4) Значения всех коэффициентов НЕ равны 0 (a!=0, b!=0, c!=0), а дискриминант меньше 0 (d0). Ожидается сообщение Нет корней
5) Значения всех коэффициентов НЕ равны 0 (a!=0, b!=0, c!=0), а дискриминант равен 0 (d=0). Ожидается вывод одного корня.
6) Значения всех коэффициентов НЕ равны 0 (a!=0, b!=0, c!=0), а дискриминант больше 0 (d0). Ожидается вывод двух корней.
